Image Compression

Difference between lossy and lossless image compression

As stated on the audio compression page, a definition of lossy compression is a process of shrinking the size of a file while loosing an amount of bits. On the other hand, lossless compression is the process of shrinking a file while retaining the same amount of bits. With images, lossy compression removes some unnoticeable pixels of colour, while lossless compression uses a different strategy.

Why use lossy or lossless files?

Lossless files are great to use when wanting to print or fax an image or keeping the quality for a different task. This way all the information is still there. while lossy image compression is good for storing personal photos on a phone or laptop or posting them on social media.

Introduction to JPEG

JPEG stands for Joint Photographic Experts Group and is a popular, widely used form of lossy compression. It removes unnoticeable pixels from the image and in the process, makes the file size smaller.
